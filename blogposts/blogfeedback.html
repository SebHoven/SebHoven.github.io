Code

In mijn code heb ik na de feedback voornamelijk class en id names veranderd zodat ze allemaal dezelfde schrijfwijze volgen. Dit betekent dat ik alles wat eerst camel case gebruikte, nu streepjes tussen de woorden gebruikt. Ook heb ik wat clutter en herhaling uit mijn code gehaald, voornamelijk de div's en plaatjes die naast elkaar stonden niet twee verschillende class names gegeven. Maar dezelfde class en voor de kleine dingen die afweken een id aangemaakt.

Website

Op de website heb ik er vooral voor gezorgt dat niet alle tekst gecentreerd was maar dat het nu voornamelijk gewoon aan de linker kant van de pagina staat. Ook heb ik mijn pagina aangepast zodat de footer (bijna) altijd helemaal zichtbaar is en dat je niet heoft te scrollen om het helemaal in beeld te krijgen. Ik heb ook mijn homepagina aangepast zodat het iets duidelijker is dat hij is gemaakt door mij en van mij is, meer identiteit dus. En ik heb de layout van de FAQ en blog pagina omgewisselt, FAQ maakt nu gebruik van het summary element omdat er meer informatie op deze pagina staat dan de blog pagina.
