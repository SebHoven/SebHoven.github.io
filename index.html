Waarom ICT?

Ik heb ervoor gekozen om ICT te studeren omdat ik heel erg van gamen houd, sinds dat ik klein ben heb ik veel spellen op de computer of op de DS gespeeld. Daarom zou ik nu graag willen weten wat er allemaal nodig is om een game te laten werken.

Ook is DougDoug een inspiratie voor mij, zijn video's zijn ontzettend vermakelijk. Maar vaak komt dit omdat hij unieke ideeën heeft die hij door middel van zijn eigen geschreven code waar kan maken. Ik hoop dat ik later ook met creatieve programmeer ideeën op kan komen en deze met de wereld kan delen. Ik hoop hiermee andere mensen gelukkig te kunnen maken.

Met het schrijven van eigen code hoop ik ook de touwen in eigen handen te krijgen zodat je niet per se van andere mensen afhankelijk bent om bijvoorbeeld mods voor je favoriete spel te maken, maar dat je dit zelf kan doen. En dit dus wederom met andere mensen zou kunnen delen.

Waarom past ICT bij mij?

  • Ik geef niet snel op, dit zorgt ervoor dat ik niet heel erg snel gedemotiveerd wordt als dat ene beetje code niet werkt of als er ergens een foutje in zit.

  • Ik ben goed in zelfstandig leren, veel van de dingen die je leert bij ICT leer je in je eigen tijd, ik leer graag nieuwe dingen dus ik heb hier geen problemen mee.

  • Ik kan goed in groepsverband werken. Een groot aantal opdrachten die we na deze opdracht krijgen worden in groepsverband gedaan,

  • In Nederland zijn alle datacenters efficiënter dan 50%, dit komt mede doordat de hitte die wordt opgewekt voor andere doel einden gebruikt kan worden.
